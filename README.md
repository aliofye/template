### React + Vite + Bun + Hono + Lefthook = ❤️

> \>_ Out there, it’s so breathtaking, isn’t it?

**Install dependencies:**  
```bash
npm install
```

**Install git hooks**
```bash
npx lefthook install
```

**Run dev server**
```bash
npm run dev
```

**Run tests**
```bash
npm run test
```

**Check test coverage**
```bash
npm run coverage
```

**Run linter**
```bash
npm run lint
```
